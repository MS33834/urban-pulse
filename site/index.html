Urban Pulse · 城市脉搏

一个想做很久的城市经济观测站。10座中国城市，16年数据，5年预测。

这是什么

毕业设计。但毕业后不想扔在那吃灰。

一直想看城市经济数据怎么变化——不只是单个城市自己的走势，而是十座城市放在一起，谁在加速、谁在减速、谁在换挡。GDP 增速放缓是暂时的还是趋势性的？哪些城市在抢人、哪些在留人？

Urban Pulse 就是干这个的。自动采集数据、跑预测、生成报表。每年1月15号自动触发一次数据更新，然后重新跑一遍全量预测。长期跑着，年复一年。

怎么用

🐳 Docker（推荐）

# 两行命令跑起来
git clone https://github.com/badhope/urban-pulse
cd urban-pulse
docker compose up
# 打开 http://localhost:8000

🐍 Python

pip install -r requirements.txt
uvicorn backend.api.main:app --reload

浏览器打开 http://localhost:8000/dashboard 看仪表盘，/docs 看 API。

数据从哪来

公开数据源，手动整理成 JSON 扔进 data/cities/。十个城市各一个文件，每个文件按年份存五个指标：

  • GDP（亿元）
  • GDP 增速（%）
  • 常住人口（万人）
  • 财政收入（亿元）
  • R&D 投入强度（%）

每年1月15号 GitHub Actions 自动触发，跑一遍采集脚本，重新训练预测模型，更新报表。长期跑着，年复一年。
