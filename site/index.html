⟁ What is Urban Pulse?

An open-source city economic intelligence platform that collects, analyzes, and forecasts urban economic indicators. Built as a graduation project, designed for long-term growth as a community-powered economic observatory.

  • 📊 Multi-city comparison — 10 major Chinese cities, side-by-side
  • 🔮 Ensemble forecasting — SARIMA, Holt-Winters, Monte Carlo simulation
  • 📈 Interactive dashboard — ECharts-powered visual analytics (run locally)
  • 🔄 Extensible pipeline — 4 plugin types: data source, analysis, forecast, visualization
  • 📅 Auto data refresh — yearly scheduled pipeline via GitHub Actions

🚀 Quick Start

🐳 Docker (recommended)

git clone https://github.com/badhope/urban-pulse
cd urban-pulse
docker compose up
→ http://localhost:8000

🐍 Native Python

pip install -r requirements.txt
uvicorn backend.api.main:app --reload
→ http://localhost:8000

⏳ Data Policy

Historical data: 2010–2025 (yearly).
Forecasts: 5-year ahead with 95% confidence intervals.
Auto-refresh: Annual pipeline triggered every January 15.
Open data: All data stored in data/cities/ as JSON.
